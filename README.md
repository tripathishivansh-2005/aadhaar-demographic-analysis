# Aadhaar Demographic Update Analysis

## Overview
This project analyses age-based and regional patterns in Aadhaar demographic updates using datasets provided by UIDAI.

## Objective
To uncover societal and administrative trends in Aadhaar demographic updates and derive insights useful for infrastructure planning and service optimisation.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Contents
- `aadhaar_demographic_analysis.ipynb` – main analysis notebook
- `*.png` – key visualisations used in the report

## Note
The dataset used is sourced from UIDAI. Full datasets are not included due to size and policy constraints.
